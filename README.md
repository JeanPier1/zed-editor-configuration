# zed-editor-configuration

## Installation

1. Install [zed](https://zed-editor.org/download.html)
2. Clone this repository
### Windows
3. Open zed and go to `File > Open Folder...` and select the `C:/Users/{username}/AppData/Roaming/Zed` folder
### Linux
3. Open zed and go to `File > Open Folder...` and select the `~/.config/zed` folder

4. Enjoy!
